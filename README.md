# Chekku

The future gem that checks your software dependencies

## Installation

Add this line to your application's Gemfile:

    gem 'chekku'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chekku

## Usage

```
  $ chekku verify
  $ chekku install (may not work before a while)
```
